# soosproject.ai

Source for [soosproject.ai](https://soosproject.ai) — the SOOS Project IETF drafts site.

Built with [VitePress](https://vitepress.dev). Deployed to GitHub Pages.

## Development

```bash
npm install
npm run docs:dev
```
